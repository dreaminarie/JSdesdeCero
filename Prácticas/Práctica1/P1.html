Ejercicio: Tipos de Datos en JavaScript

Descripción

Este ejercicio imprime en la consola los distintos tipos de datos que existen en JavaScript (number, string, boolean, undefined, null, symbol, object, array, function).

Instrucciones

Abre la consola del navegador (F12 o clic derecho → Inspeccionar → Consola) para ver los resultados.
