Ejercicio: Evaluación de Notas en JavaScript

Descripción

Este ejercicio permite ingresar una nota de algún estudiante y muestra su evaluación en función del valor ingresado.

Instrucciones

Aparecerá un prompt que te preguntará por el dato, pero para revisar el resultado abre la consola del navegador (F12 o clic derecho → Inspeccionar → Consola). Para reiniciar el prompt solo se debe de refrescar la página
